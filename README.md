# Mobile Starter 
一个移动端页面脚手架

## CSS框架
CSS框架力求精简，只提供简单的结构支持，具体样式全由使用者决定。

### Slide Menu
要使用Slide Menu，你必须将你的页面内容用一个容器包住。Slide Menu的结构像下面这样：
```html
<nav class="slide-menu right-slide-menu">
	<!-- 菜单内容 -->
</nav>
<div class="has-slide-menu has-right-slide-menu">
	<!-- 你的页面内容 -->
</div>
```
